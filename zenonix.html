How To Use
Hướng Dẫn Sử Dụng
Chỉ cần 3 bước đơn giản để dùng Zenonix Hub
01
Mở Roblox và vào game
Vào bất kỳ game Roblox được hỗ trợ. Đảm bảo bạn đã vào trong game (không phải đang loading).
02
Mở executor và paste script
Mở executor (Synapse X, KRNL, Script-Ware...). Copy script bên dưới và paste vào ô Execute.
loadstring(game:HttpGet("https://raw.githubusercontent.com/imz-yuki/zenonixhub/refs/heads/main/copyskin.lua"))()
03
Nhấn Execute và tận hưởng!
Nhấn Execute. Script sẽ tự tải từ GitHub và hiển thị UI. Console sẽ in ra thông báo ZENONIX HUB LOADING... khi thành công.
